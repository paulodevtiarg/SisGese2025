Conheça o Projeto

A evolução de uma necessidade real dentro do sistema socioeducativo do Tocantins.

Em 2017, ao assumirem o concurso do sistema socioeducativo do Tocantins, alguns servidores identificaram a necessidade de um sistema computacional para gestão das unidades, especialmente no CEIP Norte, onde muitos arquivos físicos já apresentavam sinais de deterioração, incluindo dados sensíveis dos internos.

Um dos servidores, com formação e experiência no desenvolvimento de sistemas web, propôs a criação de uma solução digital. No entanto, a ideia não foi inicialmente aceita, principalmente devido aos custos envolvidos e à falta de integração entre as equipes naquele momento.

Em 2021, durante a pandemia de COVID-19, o projeto foi novamente apresentado, desta vez de forma mais estruturada e completa. Apesar de reconhecido pela gestão, não foi aprovado, sob o entendimento de que já existia uma ferramenta disponível: o SIPIA SINASE — que, embora eficiente, não atende plenamente às demandas específicas da gestão operacional das unidades.

Em 2026, o projeto ganha força com o apoio de servidores que reconhecem a urgência de uma solução própria. Assim nasce o SYSGESE — Sistema de Gestão Socioeducativo do Tocantins, desenvolvido entre 01/01/2026 e 01/04/2026.

🚀 Atualmente, o sistema encontra-se em fase de desenvolvimento, já contando com:

  • ✔ Banco de dados online
  • ✔ Servidor em produção (Render)
  • ✔ Cadastro de adolescentes funcional
  • ✔ Visualização de dados em funcionamento
